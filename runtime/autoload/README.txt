The autoload 디렉토리는 표준 Vim autoload scripts입니다

Autoload script는 플러그인과 일반적인 용도로 사용되는 기능입니다. 함수가 호출될 때
자동으로 실행됩니다. See ":help autoload".

gzip.vim	for 압축 파일 편집
netrw*.vim	for 디렉토리 탐색 및 원격 파일 편집
tar.vim		tar files 탐색
zip.vim	  zip files 탐색
paste.vim	 mswin.vim, menu.vim 및 macmap.vim의 공통코드
spellfile.vim	 missing spell file 다운로드

Omni completion files:
ccomplete.vim		C
csscomplete.vim		HTML / CSS
htmlcomplete.vim	HTML
javascriptcomplete.vim  Javascript
phpcomplete.vim		PHP
pythoncomplete.vim	Python
rubycomplete.vim	Ruby
syntaxcomplete.vim	from syntax highlighting
xmlcomplete.vim		XML (uses files in the xml directory)
