플러그인 디렉토리는 표준 Vim 플러그인 스크립트 용입니다.

.vim으로 끝나는 모든 파일은 시작할 때 Vim에 의해 소스화됩니다.
파일을 삭제하지 않고 비활성화하는 방법에 대한 힌트를 파일에서 찾아보십시오.

getscriptPlugin.vim  Vim 스크립트 최신 버전 얻기
gzip.vim	     압축 파일 편집
matchparen.vim	     커서 아래에 일치하는 괄호를 강조 표시합니다.
netrwPlugin.vim	     네트워크를 통해 파일을 편집하고 디렉토리를 탐색 (원격)
rrhelper.vim	     --remote-wait 편집에 사용됩니다.
spellfile.vim	     누락 된 spellfile을 다운로드하십시오
tarPlugin.vim	     tar 파일 편집 (압축)
tohtml.vim	     구문 강조가있는 파일을 HTML로 변환
vimballPlugin.vim    .vba 파일 생성 및 압축 해제
zipPlugin.vim	     zip archives 편집

참고 : explorer.vim 플러그인은 더 이상 존재하지 않으며 
netrw.vim 플러그인은 디렉토리를 탐색합니다. (원격 디렉토리의 경우도 동일)
